🧊 ICE METROPOLIS - GITHUB DEPLOY INSTRUKCJA

KROK 1: STWÓRZ REPO NA GITHUB
1. Otwórz przeglądarkę w telefonie
2. Wejdź na: https://github.com
3. Zaloguj się (lub załóż konto)
4. Kliknij "+" → "New repository"
5. Wpisz nazwę: "ice-metropolis-nft"
6. Opis: "ICE METROPOLIS - NFT Dashboard System"
7. Public repository
8. Kliknij "Create repository"

KROK 2: POŁĄCZ Z TERMUX
1. Skopiuj URL repo (będzie coś jak: https://github.com/TWOJA_NAZWA/ice-metropolis-nft.git)

KROK 3: WYKONAJ W TERMUX:
git remote add origin https://github.com/TWOJA_NAZWA/ice-metropolis-nft.git
git branch -M main
git push -u origin main

KROK 4: AKTYWUJ GITHUB PAGES
1. Wróć na GitHub w przeglądarce
2. Wejdź w swoje repo "ice-metropolis-nft"
3. Kliknij "Settings" → "Pages"
4. W "Source" wybierz "Deploy from a branch"
5. W "Branch" wybierz "main" i folder "/ (root)"
6. Kliknij "Save"
7. Poczekaj 2 minuty
8. Odbierz URL (będzie: https://TWOJA_NAZWA.github.io/ice-metropolis-nft)
